计算机硬件基础 CPU · 内存 · 存储 · 外设

  • CPU：负责指令执行与运算，核心指标包括主频、核心/线程数与缓存。
  • 内存（RAM）：程序运行时的临时数据存放处，容量与频率影响并发与响应速度。
  • 存储：固态硬盘（SSD）与机械硬盘（HDD），SSD更快更适合系统与软件安装。
  • 主板与总线：连接与协调各部件的数据交换，影响可拓展性与稳定性。
  • 外设：显卡、网卡、显示器、键鼠等，针对不同场景优化。

操作系统与文件系统 Windows · Linux · macOS

  • 进程与线程：进程是资源分配单位，线程是调度单位，合理并发提升效率。
  • 内存管理：分页与分段、虚拟内存、内核态与用户态。
  • 文件系统：常见如 NTFS、ext4、APFS；目录、权限与链接是基础必修。
  • 终端与命令：掌握基本命令能显著提升效率（如 cd、ls、grep、chmod）。

计算机网络入门 IP · DNS · HTTP · TLS

网络让计算机互联并交换数据。从地址解析、路由转发到应用协议，每一层都承担清晰责任。掌握网络基础能帮助你排障、优化性能并设计可靠服务。

网络分层模型 OSI 与 TCP/IP

  • 链路层：MAC 地址与帧传输。
  • 网络层：IP 寻址与路由（如 IPv4/IPv6）。
  • 传输层：TCP（可靠）与 UDP（低延迟）。
  • 应用层：HTTP、DNS、SMTP 等。

常用网络命令 排障利器

# 测试连通性与延迟
ping example.com

# 查看域名解析
nslookup example.com

# 路由跟踪（Windows 使用 tracert）
traceroute example.com

# 简单查看 HTTP 头
curl -I https://example.com

Python 示例 脚本与数据处理

def fib(n: int) -> list[int]:
    """返回前 n 个斐波那契数列"""
    seq = []
    a, b = 0, 1
    for _ in range(n):
        seq.append(a)
        a, b = b, a + b
    return seq

if __name__ == "__main__":
    print(fib(10))

C++ 示例 性能与类型安全

#include <iostream>
#include <vector>

std::vector<long long> fib(size_t n) {
    std::vector<long long> seq;
    seq.reserve(n);
    long long a = 0, b = 1;
    for (size_t i = 0; i < n; ++i) {
        seq.push_back(a);
        long long next = a + b;
        a = b; b = next;
    }
    return seq;
}

int main() {
    auto v = fib(10);
    for (auto x : v) std::cout << x << " ";
    return 0;
}

算法与数据结构 复杂度 · 栈队列 · 树图 · 排序搜索

  • 时间/空间复杂度：用 O( ) 衡量算法随输入规模增长的资源消耗。
  • 数据结构：数组、链表、栈、队列、哈希、树与图是问题建模的基石。
  • 经典算法：排序（快排、归并）、搜索（二分）、最短路径（Dijkstra）。
# Python：二分查找
def binary_search(nums, target):
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            return mid
        if nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1

安全与最佳实践 密码学 · 输入校验 · 最小权限

  • 输入校验与过滤：防止注入与越权访问，白名单策略优先。
  • 认证与授权：区分身份识别与权限分配，确保数据与操作边界。
  • 加密与传输安全：使用 TLS 保障传输安全，敏感数据静态加密与密钥管理。
  • 日志与审计：为定位问题与合规留痕，注意隐私与最小留存。
